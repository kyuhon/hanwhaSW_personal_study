NVL(a,b)
	null인 경우 b 반환, 아니면 a 반환

NVL2(a,b,c)
	a가 null이 아니면 b 반환 , null이면 c 반환

coalesce(a,b,c)
	앞에부터 null값이 존재하면 그다음값이 반환
	전부 null이면 null 반환

NULLIF(인수1, 인수2) 
인수1과 인수2가 같으면 NULL을 반환하고 같지 않으면 인수1을 반환해주는 함수

SIGN은 해당 숫자가 양수면 1, 음수면 -1, 0이면 0을 출력

DECODE(A, B, X, Y) A = B 이면 X를 출력, A ≠ B 이면 Y를 출력
DECODE(A, B, X, C, Y, Z)
	A = B이면 X 출력,
	A = C이면 Y 출력,
	A ≠ B 이고 A ≠ C이면 Z 출력

SUBSTR
	SELECT SUBSTR('HELLO WORLD', 1, 5) FROM DUAL;  -- HELLO
	SELECT SUBSTR('HELLO WORLD', 7) FROM DUAL;     -- WORLD

REPLACE
	SELECT REPLACE('HELLO WORLD', 'L', 'X') FROM DUAL; -- HEXXO WORXD
	SELECT REPLACE('HELLO WORLD', 'L') FROM DUAL;      -- HEO WORD

CONCAT
	SELECT CONCAT('HELLO', ' WORLD') FROM DUAL; -- HELLO WORLD
	SELECT 'HELLO' || ' ' || 'WORLD' FROM DUAL; -- HELLO WORLD
