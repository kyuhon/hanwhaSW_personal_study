11.25 렌탈브레인 회의

프로그램을 사용하는 사람은 소기업인 경우 사장이거나 마케터이다.

사장과 마케터들은 각자의 회사마다 전부 다른 요구사항을 원할테고
사용하는 사람이 필요로 하는 기능들이 무조건 있어야한다.

구체적인 고객의 범위를 정하고 업체에 대한 정의를 내리기

구체적인 고객이 없이 모든 대상을 하면 이도저도 아닌 CRM이 만들어질거같은데..




정리

채널 - 데이터분석 - 캠페인
CRM은 데이터를 분석해 맞는 고객을 선정하고, 선정된 고객에게 적절한 채널로 적절한 메시지를 자동으로 보내는 시스템
	채널 = 어떻게 보낼까
	분석  = 누구에게 보낼까
	캠페인 = 무엇을 보낼까 + 언제 보낼까


채널
고객에게 도달하는 메시지 경로


데이터 분석
고객 행동 분석 - > 인사이트 -> 액션


캠페인
분석된 데이터를 기반으로 특정 고객군에게 자동 또는 수동으로 메시지를 발송하는 마케팅/운영 액션





전자결제의 범위
견적서 승인 이후 고객 데이터 생성
이 때 견적서에 들어가는 내용

전자서명
	계약에 대한 동의

	고객이 계약금/ 선착금/ 초기 비용을 전자결제로 납부하며 승인
